# Road Fighter

Dynamic, arcade-style racing game where you take the role of a highway driver aiming to survive as long as possible by avoiding collisions and obstacles.

## About the Project
The primary objective of the project was to develop a game titled **Road Fighter**. The gameplay focuses on quick reflexes and high scores, offering an exhilarating experience for players.

### Key Objective:
- Maintain control on a highway while dodging other vehicles and obstacles.
- Aim to achieve the highest possible score.

## Getting Started

### Prerequisites
- Windows Operating System
- [SFML (Simple and Fast Multimedia Library)](https://www.sfml-dev.org/)
- Necessary DLL files included in the release.

### Installation
1. Download the game files.
2. Run `Road-Fighter.exe` to start the game.

## How to Play
1. **Launch the Game:** Open `Road-Fighter.exe`.
2. **Main Menu:** 
   - Enter your nickname (up to 8 characters).
   - Navigate using arrow keys and press `Enter` to confirm.
   - Available options: `Start Game`, `Highest Score`, `Tutorial`, `About`, and `Exit`.
3. **Gameplay:** 
   - Steer your car using arrow keys to avoid obstacles.
   - Collect life-increasing power-ups.
   - Earn points for every second you survive.
4. **Game Over:** Check your score and try again or exit.

## Features
- **Main Menu:** Navigate through game options including tutorial and high scores.
- **Power-Ups:** Enhance gameplay with additional life bonuses.
- **Score Tracking:** Save and display the highest scores using JSON storage.
- **Dynamic Gameplay:** Randomized enemy cars and real-time collision checks.

## Technology Stack
- **Languages:** C++
- **Libraries:** 
  - [SFML](https://www.sfml-dev.org/) for graphics, sounds, and window handling.
  - [nlohmann::json](https://github.com/nlohmann/json) for data storage.
- **C++ Features:** Modules, regular expressions, filesystem, and ranges.
- **Development Tools:** Visual Studio, Debugging tools for runtime DLL management.
